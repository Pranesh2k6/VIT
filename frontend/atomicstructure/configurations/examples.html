Example Problems: Configurations

Practice filling those electron shells!

⭐ Example 1: Writing Configuration

Q: Write the ground state electronic configuration for Phosphorus (P, Z=15).

✅ Solution:
Order of filling: 1s, 2s, 2p, 3s, 3p...
Fill with 15 electrons respecting Pauli and Hund:
$1s^2$ (2 e⁻) $\rightarrow$ $2s^2$ (4 e⁻) $\rightarrow$ $2p^6$ (10 e⁻) $\rightarrow$ $3s^2$ (12 e⁻) $\rightarrow$ $3p^3$ (15 e⁻).
Configuration: $\mathbf{1s^2 2s^2 2p^6 3s^2 3p^3}$ (or [Ne] 3s² 3p³)

⭐ Example 2: Hund's Rule Application

Q: How many unpaired electrons are present in a neutral Nitrogen atom (N, Z=7)?

✅ Solution:
Configuration: $1s^2 2s^2 2p^3$.
The $1s$ and $2s$ orbitals are filled ($\uparrow\downarrow$).
According to Hund's rule, the 3 electrons in the $2p$ subshell occupy the three degenerate $p$ orbitals singly with parallel spins ($\uparrow$ $\uparrow$ $\uparrow$).
Therefore, there are $\mathbf{3}$ unpaired electrons.

⭐ Example 3: Exception Configuration

Q: Write the ground state electronic configuration for Copper (Cu, Z=29), noting the exception.

✅ Solution:
Expected (Aufbau): $[Ar] 4s^2 3d^9$.
Actual (Exception): To achieve a more stable fully-filled d subshell, one electron moves from 4s to 3d.
Configuration: $\mathbf{[Ar] 4s^1 3d^{10}}$
