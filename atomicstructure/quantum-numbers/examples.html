Example Problems: Quantum Numbers

Practice assigning electron addresses!

⭐ Example 1: Possible 'l' values

Q: If the principal quantum number $n = 4$, what are the possible values for the azimuthal quantum number $l$?

✅ Solution:
The value of $l$ ranges from $0$ to $(n-1)$.
For $n=4$, the possible $l$ values are $\mathbf{0, 1, 2, 3}$. (These correspond to s, p, d, and f subshells).

⭐ Example 2: Possible 'm_l' values

Q: If the azimuthal quantum number $l = 2$ (a d subshell), what are the possible values for the magnetic quantum number $m_l$?

✅ Solution:
The value of $m_l$ ranges from $-l$ to $+l$, including 0.
For $l=2$, the possible $m_l$ values are $\mathbf{-2, -1, 0, +1, +2}$. (This confirms there are 5 d orbitals).

⭐ Example 3: Identifying Invalid Sets

Q: Which of the following sets of quantum numbers ($n, l, m_l, m_s$) is NOT allowed for an electron in an atom?

a) (3, 2, -1, +1/2)    b) (2, 2, 0, -1/2)    c) (4, 0, 0, +1/2)    d) (1, 0, 0, -1/2)

✅ Solution:
Rule: $l$ must be less than $n$ ($l = 0, ..., n-1$).
In option (b), $n=2$ but $l=2$. Since $l$ cannot be equal to $n$, set (b) is **not allowed**.
